# Alfred University Computer Science Program Website

Welcome to the official website for the **Computer Science program at Alfred University!** The site provides a clean summary of the curriculum, faculty, research opportunities, and more within the department.


## ✔️ Purpose

This website is designed to:
- Provide a clear overview of the CS curriculum
- Showcase research opportunities
- Highlight resources available to students
- Introduce the faculty
- Help students navigate the program


## ⭐ About the Program

The Computer Science program at Alfred University focuses on building strong foundations in:
- Software development
- Algorithms and data structures
- Networking
- Theory

The department also offers unique programs in:
- AI & Machine Learning
- Data Science
- Cybersecurity


## 🖥️ Running This Site Locally (using Docker)

Navigate to your local repository folder in a terminal and run:

```bash
docker build -t site .
docker run -p 5000:5173 site:latest
```

And navigate to:

```
localhost:5000
```
